# Neovim shortcuts

- to change theme : `space+t+h`

## Shortcuts for nvim
- to open filetree : `ctrl+n`
- to mark/unmark file : `m`
- create a new file : `a`
- copy file : `c`
- paste file : `p`
- rename file : `r`
- delete file : `d`


## Shortcut for file search
- find file : `space + f + f`
- To open cheatsheet for commands : `space+c+h`
